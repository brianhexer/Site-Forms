Basic Site-Forms Example

The simplest way to add forms to your site

Step 1: Create HTML Form

<form id="my-form" class="site-form" data-form-name="my-form">
  <input type="text" name="name" required>
  <input type="email" name="email" required>
  <button type="submit">Submit</button>
</form>

Step 2: Include Scripts

<script src="../js/form-model.js"></script>
<script src="../js/form-handler.js"></script>

Step 3: Configure Email (Choose One)

Option A: EmailJS (Browser-based)

<script src="https://cdn.jsdelivr.net/npm/@emailjs/browser@3/dist/index.min.js"></script>
<script>
  window.EMAILJS_PUBLIC_KEY = 'YOUR_KEY';
  window.EMAILJS_SERVICE_ID = 'YOUR_ID';
  window.EMAILJS_TEMPLATE_ID = 'YOUR_ID';
</script>

Option B: Custom Webhook

<form data-webhook-url="https://your-server.com/api/submit">
  ...
</form>

Common Data Attributes

Attribute Description
data-form-name Unique form identifier
data-recipient Email to receive submissions
data-redirect-url URL to redirect after submit
data-success-message Message shown on success
data-error-message Message shown on error
